Include elements provide a powerful way to build templates from other templates. An Include's functionality can be enhanced by four additional elements - Fillin, Show, Rename & Remove - which are always nested inside of an Include. Note that these nested elements do not alter the original template that is being included; they alter the copy that is made for inclusion, so that a single template can be included many times (often into many others) and used differently each time.
