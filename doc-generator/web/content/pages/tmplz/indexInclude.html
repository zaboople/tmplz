Tmplz is a Java API for producing text from templates. It is especially useful for generating HTML in web applications, but can also be used to generate XML, SQL, Javascript or any other text-based language. The design goals of Tmplz were to:

  • Treat templates as data files rather than computer programs;
  • Allow templates to be built from other templates flexibly, to maximize reuse & avoid repetition;
  • Be small, fast, and efficient;
  • Integrate easily into applications, web or otherwise.

This documentation was generated using Tmplz.

System Requirements

Tmplz requires Java, version 1.6 or greater. It can be used in web applications, but does not require Servlets or any kind of web server.
