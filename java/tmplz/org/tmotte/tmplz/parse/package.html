For internal use. This contains the classes that Tmplz uses to parse html template files. There are four steps in parsing a template:
  1. The template is converted from text to Tokens by the DocumentTokenizer, in the tokenize package.
  2. The tokens are converted into "dum" nodes by the DocumentBuilder.
  3. The dum nodes are "preprocessed" by the Preprocessor.
  4. The dum nodes are turned into a Section by the TemplateBuilder.
