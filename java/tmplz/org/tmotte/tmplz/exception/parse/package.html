Exceptions in this package are caused by syntax errors in templates. The inheritance hierarchy of exceptions is:
  • AbstractParsingException
    • TemplateNestedException: Always thrown when a template cannot be parsed. Provides the name of the template. Always contains another Exception, either a TemplateContentException or another TemplateNestedException.
    • TemplateContentException: Abstract class for all other parsing Exceptions.
      • IncludeNestedException: Thrown when an Include fails because of errors in an included template. Always contains an Exception that describes the actual syntax error.
      • All the other Exceptions. These do not have any "nesting" and describe a specific syntax error.
